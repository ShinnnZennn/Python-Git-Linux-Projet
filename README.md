## Python · Git · Linux Project

### Project Description

This project is an online quantitative finance dashboard developed with Python and Streamlit.
It retrieves financial market data, applies quantitative strategies, and displays results through an interactive web interface.
The application is deployed on a Linux server and accessible online.

---

### Live Application

```
http://13.60.51.151:8501/
```

---

### Features

* Real-time financial data retrieval
* Automatic refresh every 5 minutes
* Single asset analysis (Quant A)
* Multi-asset portfolio analysis (Quant B)
* Interactive parameters and visualizations
* Daily report generated automatically using a cron job

### Deployment

* Hosted on a Linux AWS EC2 instance
* Application runs continuously using tmux
* Public access via IP and port

### Technologies

Python, Streamlit, Pandas, NumPy, yFinance, GitHub, Linux, AWS
